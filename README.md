# ✉️ Carta Misteriosa (Meme Letter Generator)

Uma aplicação web divertida e criativa que transforma frases comuns em "cartas misteriosas" estilizadas — inspiradas na estética clássica de bilhetes de recortes de jornais e revistas de filmes antigos. Cada palavra ganha sua própria identidade visual dinamicamente.

---

## 🚀 Funcionalidades

Com base na interface apresentada (`image_e3b26c.png`), o projeto conta com as seguintes mecânicas:

* **Geração Dinâmica:** Digite uma frase no campo de texto e clique em **"Criar Carta"**.
* **Estilização Aleatória:** O JavaScript intercepta o texto, divide a frase palavra por palavra e atribui de forma randômica uma cor de fundo, cor de texto, tamanho e família de fonte diferente para cada bloco.
* **Contador de Palavras:** O sistema analisa a string enviada e exibe no topo a contagem exata de palavras geradas na tela (como o "4" visível no exemplo do texto *"testando A carta MISTERIOSA"*).

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura dos inputs, botões e do container onde o bilhete recortado é renderizado.
* **CSS3:** Definição das múltiplas classes de fontes, espaçamentos (*paddings*) e alinhamento dos blocos estilo "recorte".
* **JavaScript (ES6):** Manipulação de strings (`split`), lógica de sorteio aleatório de estilos e inserção dinâmica de elementos no DOM.
